## Clone this repo

```
git clone git@github.com:fandredev/nlw-connect.git
```

## Go to directory

```
 cd nlw-connect
```

## Creating a python environment

```
 python -m venv venv
```

## Install dependencies

```
 pip install -r requirements.txt
```

## Run integration tests

```
 pytest
```

## Run server

```
 python run.py
```
